# posturecorrection
# Posture Detection Project

##  Description

This project detects human posture using a webcam and helps identify whether the posture is correct or incorrect.



## Features

* Real-time posture detection
* Detects incorrect posture
* Simple and easy to use



##  Technologies Used

* Python
* OpenCV
* MediaPipe



##  How to Run

1. Install required libraries:
   pip install -r requirements.txt

2. Run the program:
   python main.py


##  Project Files

* main.py
* requirements.txt
* other supporting files



## Output

The system uses the webcam to track body posture and shows whether posture is correct or not.
